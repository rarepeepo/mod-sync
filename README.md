# Mod Sync
A mod for minecraft that makes sure you always have the latest mods to play on your friend's server.

## Building
Clone the repo with:
```
git clone https://github.com/rarepeepo/mod-sync
```
Enter the newly created directory and run:
```
./gradlew build
```
